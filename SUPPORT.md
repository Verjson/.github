# Support

Thanks for using verJSON projects! Here's how to get help.

## 📚 Documentation

Start with the project's `README.md` and any docs linked from it. Many common
questions are answered there.

## 💬 Questions & Discussion

For general questions, ideas, and conversation, use
[GitHub Discussions](https://github.com/orgs/Verjson/discussions).
This is the best place to reach the community and the team.

## 🐛 Bugs & Feature Requests

- Found a bug? Open an issue using the **Bug Report** template.
- Have an idea? Open an issue using the **Feature Request** template.

Please search existing issues first to avoid duplicates.

## 🔒 Security Issues

Do **not** report security vulnerabilities in public. Follow our
[Security Policy](./SECURITY.md) instead.

## 🌐 More

Learn more about what we're building at [verjson.com](https://verjson.com).

> Note: GitHub issues are for bugs and feature requests, not general support
> questions. Please use Discussions for those so issues stay actionable.
